## Fibonacci Number Again - F(n) mod m - Pisano Period

```


-----

PROBLEM: Given two integers n and m, output F(n) mod m (that is, the remainder of Fn when divided by m).

INPUT: The input consists of two integers n and m given on the same line (separated by a space).

CONSTRAINTS: 1 <= n <= 10^18;  2 <= m <= 10^5

OUTPUT: Output F(n) mod m.

------

SAMPLE

Input: 239 1000
Output: 161

Input: 2816213588 30524
Output: 10249
-----

EXPLANATION

(F(n), m) => (239, 1000) = 161
F(239) mod 1000 = 39679027332006820581608740953902289877834488152161 (mod 1000) = 161

```
