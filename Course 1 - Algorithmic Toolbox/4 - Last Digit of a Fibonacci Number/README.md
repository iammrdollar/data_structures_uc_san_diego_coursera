## Last Digit of a Large Fibonacci Number

```

Fibonacci Sequence is defined by :

F0 = 0;
F1 = 1;

F(n) = F(n-1) + Fi(n-2); [ For all n >= 2 ]

-----

PROBLEM: Given an integer n, find the last digit of the nth Fibonacci number Fn (that is, F(n) mod 10).

INPUT: The input consists of a single integer n.

CONSTRAINTS. 0 <= n <= 10^7

OUTPUT: Output the last digit of F(n)

------

SAMPLE

Input: 200
Output: 5

Input: 331
Output: 9

-----

EXPLANATION

F(200) = 280 571 172 992 510 140 037 611 932 413 038 677 189 525
Last Digit = 5

F(331) = 668 996 615 388 005 031 531 000 081 241 745 415 306 766 517 246 774 551 964 595 292 186 469
Last Digit = 9

```
