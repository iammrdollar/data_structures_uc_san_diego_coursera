## Last Digit of the Sum of Fibonacci Numbers

```

PROBLEM: Given an integer n, find the last digit of the sum F(0) + F(1) + ... + F(n).

INPUT: The input consists of a single integer n.

CONSTRAINTS. 0 <= n <= 10^14

OUTPUT: Output the last digit of F(0) + F(1) + ... + F(n)

------

SAMPLE

Input: 3
Output: 4

Input: 100
Output: 5

-----

EXPLANATION

Sum(F(0) + ... F(3)) = 4
Last Digit = 4

Sum(F(0) + ... F(100)) = 927 372 692 193 078 999 175
Last Digit = 5

```
