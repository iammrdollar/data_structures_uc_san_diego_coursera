## nth Fibonacci Number

```

Fibonacci Sequence is defined by :

F0 = 0;
F1 = 1;

F(n) = F(n-1) + Fi(n-2); [ For all n >= 2 ]

-----

Problem: Given an integer n, find the nth Fibonacci number Fn.

Input: The input consists of a single integer n.

Output: Output Fn

------

Sample

Input: 10
Output: 55

```
